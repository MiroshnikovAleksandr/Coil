В данном файле мы считаем сопротивление всего контура, а также его длину.

Для рассчёта длин круглого, прямоугольного и кусочно-линейного контуров созданы 3 функции, работающие по одному принципу.

На входе в функцию мы получаем уже оптимизированный многомерный массив, первое измерение данного массива - витки,
соединённые параллельно, а второе измерение - витки, соединённые последовательно.

Для круглой катушки: (coils - двумерный массив)

Пример: coils = [[r1, r2], [r3, r4], [r5, r6]], где ri - радиус i-ого витка

С помощью цикла мы для каждого витка, соединённого параллельно, суммируем радиусы витков, соединённых последовательно.

Для прямоугольной катушки: (coils - трёхмерный массив)

Пример: coils = [[[X_side_1, Y_side_1], [X_side_2, Y_side_2]], [[X_side_3, Y_side_3], [X_side_4, Y_side_4]], [[X_side_5, Y_side_5], [X_side_6, Y_side_6]]],
где [X_side_i, Y_side_i] - стороны i-ого прямоугольного витка

С помощью цикла мы для каждого витка, соединённого параллельно, суммируем 2 стороны прямоугольника (X_side, Y_side),
а затем домножаем на 2.

Для кусочно-линейной катушки: (coils - четырёхмерный массив)

Пример: coils = [[[[x1_1, y1_1], [x2_1, y2_1], [x3_1, y3_1]], [[x1_2, y1_2], [x2_2, y2_2], [x3_2, y3_2]]],
[[[x1_3, y1_3], [x2_3, y2_3], [x3_3, y3_3]], [[x1_4, y1_4], [x2_4, y2_4], [x3_4, y3_4]]]],
где [[x1_i, y1_i], [x2_i, y2_i], [x3_i, y3_i]] - координаты i-ого витка (в данном случае треугольника)

С помощью цикла мы для каждого витка, соединённого параллельно, и для каждого витка, соединённого последовательно
находим длину 1 отрезка 1 витка, а затем суммируем их.
